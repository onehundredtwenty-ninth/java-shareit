# java-shareit

Backend-приложение на Java для сервиса аренды и совместного использования вещей.

## Основной функционал

В приложении реализованы следующие возможности:

- управление пользователями
- создание и редактирование вещей
- поиск вещей по текстовому запросу
- создание бронирований
- подтверждение и отклонение бронирований владельцем вещи
- получение истории бронирований
- создание запросов на вещи
- получение списка запросов и связанных с ними вещей

## Архитектура

Проект разделен на несколько модулей:

- `gateway` - слой входа в приложение, отвечающий за валидацию запросов и маршрутизацию
- `server` - основной backend с бизнес-логикой, доступом к данным и REST-контроллерами

## Технологии

В проекте используются:

- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Docker
- Maven
